Privacy & Data Handling

Vincio Chat is designed for temporary, ephemeral communication with minimal data persistence. All conversations exist only in memory and are completely deleted when rooms are closed or inactive. We use end to end encryption to ensure that no third parties can access your messages.

Minimal Data Storage

Vincio Chat does not retain chat content, messages, or user identities beyond the duration of the session. All chat data exists only temporarily in the server's memory with no persistent storage.

We use third-party services for analytics and hosting as detailed below, but have configured these services with privacy as a priority. No personally identifiable information from chat sessions is ever shared with these services.

Data Deletion Upon Room Closure

When a room is deleted, all associated messages and user data are permanently erased and cannot be recovered by any means. All chat rooms are designed to be ephemeral by default.

Session and Room Lifetime

A room's lifespan is determined by its users. The room and its messages exist only while users are connected. Rooms are automatically deleted after a period of inactivity, ensuring no residual data remains.

We use End to End Encryption aswell as token verification on all communications and data in transit.

No User Authentication

Vincio Chat requires no user accounts, Email, Fingerprint, or Password, or any personally identifiable information. Users are identified solely by temporary session IDs and room codes that exist only in memory.

Analytics

We use minimal analytics to understand simple usage patterns and improve our service.

Cloudflare Analytics

Vincio Chat uses Cloudflare Analytics in its most minimal configuration. This helps us understand basic usage patterns without identifying individual users. The analytics we collect:

  • Do not track individual users across sessions
  • Do not record IP addresses in an identifiable format
  • Do not use cookies for tracking purposes
  • Do not record any chat content or message data
  • Only collect aggregated metrics like page views, and browser types.

These analytics help us improve the stability and performance of our service without compromising privacy.

Analytics Controls

If you wish to reduce analytics collection further, we recommend using browser privacy extensions that block analytics scripts or using privacy-focused browsers.

Cloudflare's privacy policy can be found at https://www.cloudflare.com/privacypolicy/.

Hosting Provider

Information about our hosting infrastructure and related privacy implications.

Render Hosting

Vincio Chat is hosted on Render, a privacy-focused cloud hosting provider. Render may collect certain technical information as part of their standard operation, including:

  • Browser type and version
  • Time and date of connections
  • Technical metrics about server performance

Importantly, Render does not have access to the of the content inside of rooms or user identities beyond temporary connection data required for basic service delivery.

Render's privacy policy can be found at https://render.com/privacy.

Server Security

Our hosting configuration is designed with security as a priority. We implement industry standard security measures including:

  • TLS encryption for all data in transit
  • Regular security updates and patches
  • Restricted server access and permissions
  • Automatic memory management to prevent data leakage
