Privacy & Data Handling

Vincio Chat is designed for ephemeral, privacy-first communication. All chat data is encrypted and exists only in server memory and is permanently deleted when rooms close or the server restarts.

Zero Data Retention

Vincio Chat does not store messages logs, user identities, or message content in any persistent storage. There is no database, no file storage, and no backups. All data is erased immediately when a room is deleted or the server restarts.

Data Deletion Upon Room Closure

When a room is deleted or becomes inactive, all associated messages and user data are permanently erased from memory and cannot be recovered.

Session and Room Lifetime

Rooms and messages are in memory only and exist only while users are connected and active. Inactive rooms are automatically deleted after a short period, ensuring no residual data remains.

No User Authentication

Vincio Chat does not require user accounts, email, passwords, or any personally identifiable information. Users are identified only by temporary session IDs and room codes, which exist in memory for the duration of the session.

End-to-End Encryption

All messages in Vincio Chat are temporary in memory and are end-to-end encrypted in your browser using AES-GCM. Encryption keys never leave your device and are never revealed to the server.

Encryption Details

  • Messages are encrypted in the browser before being sent to the server.
  • The server only relays encrypted data and cannot decrypt message content.
  • Room codes can include the encryption key for secure sharing with other users.
  • Even if the server is compromised, your messages cannot be read without the key.

Analytics

Vincio Chat does not use any tracking, advertising, or behavioral analytics. No cookies or persistent identifiers are used for analytics or marketing.

Minimal Technical Metrics

We may collect basic, non-identifiable technical metrics (such as aggregate connection counts or error rates) for the sole purpose of monitoring server health and preventing abuse. No message content, user identities, or IP addresses are stored or analyzed for profiling.

Hosting Provider

Vincio Chat is hosted on privacy-focused cloud infrastructure. The hosting provider may collect standard technical information (such as connection times and browser types) for operational purposes, but has no access to message content or user identities.

Server Security

  • Encryption for all data in transit
  • Regular security updates and patches
  • Restricted server access and permissions
  • Automatic memory management to prevent data leakage
