Privacy & Data Handling

Scratchy Chat is designed for temporary, ephemeral communication with no data persistence. All conversations exist only in memory and are completely deleted when rooms are closed or inactive.

No Permanent Data Storage

Scratchy Chat does not retain any information from chat rooms, users, or messages beyond the duration of the session. All data exists only temporarily in the server's memory with no persistent storage.

No third-party services are used to store or process user data. The application ensures all interactions are transient and ephemeral, with zero long-term data retention.

Data Deletion Upon Room Closure

When a room is deleted, all associated messages and user data are permanently erased and cannot be recovered by any means. All chat rooms are designed to be ephemeral by default.

Session and Room Lifetime

A room's lifespan is determined by its users. The room and its messages exist only while users are connected. Rooms are automatically deleted after a period of inactivity, ensuring no residual data remains.

No User Authentication

Scratchy Chat requires no user accounts, authentication, or personally identifiable information. Users are identified solely by temporary session IDs and room codes that exist only in memory.

No Cookies or Tracking

Scratchy Chat does not use cookies, web beacons, or any form of user tracking. No browsing data, device identifiers, or session analytics are collected or stored.

Local User Configuration

All user preferences (themes, settings, etc.) are stored only in your local browser and are never logged, stored, or processed on our servers.
